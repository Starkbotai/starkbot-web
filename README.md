## Stack

- **React** + **TypeScript** + **Vite**
- **Tailwind CSS** for styling
- **D3.js** for the interactive mind map demo
- **anime.js** for animations
- **React Router** for page routing

## Getting Started

```bash
npm install
npm run dev
```

Dev server starts at `http://localhost:3000`.

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Project Structure

```
src/
  components/    # Reusable UI components (Hero, ChatDemo, MindMapDemo, etc.)
  views/         # Page-level components (Home, Docs, BrandKit)
  hooks/         # Custom React hooks
  config/        # Configuration files (chat demo sequences)
  assets/        # CSS and static assets
public/          # Static files (images, fonts)
```
